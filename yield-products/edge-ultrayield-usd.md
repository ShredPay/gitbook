# Edge UltraYield USD

#### About this vault

This Earn vault generates yield by supplying USDC into Morpho, using a higher-yield strategy curated by Edge Capital.

#### What is Morpho?

Morpho is an on-chain lending protocol where depositors supply assets to earn interest from borrowers who borrow against over-collateralized collateral.
